## How to cite

If you use these data please cite
- the original source
  > Edwards, Owen D. 2020. Metathesis and unmetathesis in Amarasi. Berlin: Language Science Press. doi: 10.5281/zenodo.3700413.
- the derived dataset using the DOI of the [particular released version](../../releases/) you were using

## Description


This dataset provides contemporary, non-overlapping language ranges for language groups of Timor and self-identified varieties of Meto as depicted in Fig. 1.1 and 1.2 of Edwards 2020.
